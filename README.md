# Chat App

A sleek, responsive Flutter chat application featuring a modern UI with distinct incoming and outgoing message bubbles.

## Features
- **Modern Chat Interface**: Clearly separated incoming and outgoing message bubbles.
- **Auto-Reply Bot**: Built-in simulated responses to test the chat flow immediately.
- **Cross-Platform Ready**: Scales cleanly across mobile, tablet, desktop, and web.
- **Material 3 UI**: Polished aesthetic with dynamic color themes.

## Getting Started

To run the app locally:
```bash
flutter pub get
flutter run
```
